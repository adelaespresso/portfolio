# adela — portfolio

Minimalistický vícestránkový web (Helvetica, černá, světlé pozadí). Fotky jsou lokálně v `assets/` s původními názvy. Home i Contact už používají vlastní fotky (nezávislé na Wixu); z Wixu se teď načítá jen sekce **creative**.

## Stránky
- **home** – dvě fotky (main1, main2) přes celou šíři
- **film / videos / photography** – přehledy (náhled = hlavní fotka sady, proklik na detail); pod fotkami názvy
- **detail filmu** – velká hlavní fotka → název + popis → ostatní stills (zarovnané do mřížky)
- **detail videa** – YouTube přehrávač (nostalgia, they call it summer, Wien) nebo velká fotka, pod tím stills, bez popisků
- **detail fotografie / moments** – galerie sady
- **creative / contact**

Galerie: fotky jsou zarovnané do pravidelné mřížky (řádky za sebou), kliknutím se otevře plná fotka + listování (←/→, Esc).

## Publikace na GitHub Pages
Stáhni `adela-portfolio-site.zip`, rozbal a nahraj obsah do repozitáře `portfolio` (GitHub Desktop: zkopíruj → Commit → Push; nebo přes web přetáhni všechny `.html` + složku `assets` + `.nojekyll`). Pak Settings → Pages → main / (root). Po nahrání Ctrl/Cmd + Shift + R.

> Pozn.: `people` se přejmenovalo na `moments` (nový soubor `moments.html`, starý `people.html` smazán). V hlavním menu (home/film/videos/photography/creative/contact) se „people" nevyskytovalo, takže tam není co měnit; „moments" je vidět jako sekce na stránce photography.
